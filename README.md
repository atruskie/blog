
# Anthony's blog

Uses Hugo and Webpack

## Install instructions

1. Get Hugo
  - Windows: `> choco install hugo`
1. Install dependencies
  - `yarn install`

##  Dev instructions

Run Dev Server (both webpack and hugo):

```
> yarn start
```

## Prod instructions

```
$ npm run build:production
```

## Editing

### Make a new post:

```
hugo new posts/name-of-post.md
```
